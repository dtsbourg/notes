Keywords: ACO, PSO, ABC; application to networking (e.g., routing), robotics (e.g.,control design and optimization), operational research (e.g., TSP)

Problem example: given a TSP problem (weighted graph), generate a solution using the desirability only, using the pheromones only, with/without elitism, with/without local search, with/without one of the rules of ACS, etc. -> you will have to crunch out a series of cities the ant(s) have gone through with their distances.

  • desirability = visibility (\(\eta_{ij} = 1 / d_{ij}\)). To build this model use the AS algorithm but remove the pheromon dependence in the transition rule (e.g. $p_{ij}^k(t) = $), no pheromone trail update.
  • pheromone: see notes.
  • without elitism: AS
  • with elitism: EAS (extra amount for the best tour), ACS (global update by best ant)
  • local update: start from some initial solution and try to improve by local changes. ACS: ant can update the amount of pheromones between i and j. Favors exploration of not yet explored edges, avoids stagnation and convergence to common path.
  • ACS also has a candidate list (cities to be visited from a given city)
  • local search: CS-k-Opt (takes a path and edits it in place to see if better result through local permutation). Less computational cost but lack of good starting solutions.
