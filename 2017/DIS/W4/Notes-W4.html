Indoor

  • Overhead Camera Systems

    • Absolute positions
    • active / passive / markerless
    • Issues : light, calibration

  • MCS (Multicamera Capture Systems)

    • mm accuracy - 100-500 Hz update (2ms latency)
    • 6D pose estimation
    • 4-5 passive markers per object needed

  • IR-UWB (Impulse Radio Ultra Wide Band)

    • Time of flight sensor for UWB tags that sync to a receiver
    • Absolute positions
    • 6-8 GHz
    • Good performance (except if interferences, multi-path, non LOS)

  • Infrared (IR) + RF

    • IR LED + photodiode
    • 3.5 m range - 25 Hz with 10 neighbours (250Hz with 2)
    • Accuracy is ~

    • 200-300 ms latency (max 5Hz update) - a few meters accuracy

Robot localization using proprioceptive sensors with uncertainties

Error sources

  • Deterministic : Can be fixed by calibration
  • Non-Deterministic : Probabilistic description

Methods for handling uncertainties

Odometry error is cumulative. We need to model these errors (e.g. model the uncertainty by Gaussian).

Non deterministic Uncertainties in Odometry based on wheel encoders

  • Variation of the contact point of the wheel
  • Unequal floor contact (e.g., wheel slip, non planar surface)

Error types :

  • Range error: sum of the wheel movements
  • Turn error: difference of wheel motion
  • Drift error: difference between wheel errors lead to heading error

Belief Representation

Gaussian

  • Advantages:

    • Compact (only mean and variance required)
    • Continuous
    • Powerful tools (KalmanFilter)

  • Disadvantages:

    • Requires Gaussian noise assumption
    • Uni-modal
    • Cannot represent ignorance (âkidnapped robot problemâ)

Particle Filter

  • Advantages:

    • Can model arbitrary beliefs
    • No assumptions on noise characteristic

  • Disadvantages:

    • No unique solution
    • Not continuous
    • Computationally expensive
    • Tuning required

Take-Home messages

  • There are several localization techniques for indoor and outdoor systems
  • Each of the localization methods/positioning system has advantage and drawbacks
  • Odometryis an absolute positioning method using only proprioceptive sensors but affected by a cumulative error
  • Localization error can be modeled and estimated
  • The error propagation methods/filtering techniques are tools applicable to a large variety of noisy problems, not just localization and navigation
  • Feature-based navigation is a way to compensate odometrylimitations
